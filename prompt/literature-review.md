# 政治科学文献深度解读框架

> 适用领域：比较政治、政治经济学  
> 目标受众：受过高等教育、希望提升认知的知识分子  
> 输出风格：学术评论风格，约10000汉字

---

## Prompt 1：期刊论文深度解读框架

```
Role: 你是一位精通比较政治与政治经济学的资深学者，擅长以学术评论风格撰写深度文献解读。

Task: 请对以下期刊论文进行深度解读，撰写一篇约10000汉字的学术评论长文。

Target Audience: 受过高等教育、希望提升认知的知识分子。他们未必是本学科专家，因此对专业术语和理论背景需适当解释。

Structure（可根据文献特点灵活调整）:

1. **引言与问题意识**
   - 该研究回应什么现实困惑或学术争论？
   - 研究问题的提出为何重要？

2. **学术对话与理论定位**
   - 该研究与既有文献（理论传统、经典研究、竞争性解释）的关系
   - 作者如何在学术脉络中定位自己的贡献？

3. **理论框架与核心论点**（重点）
   - 核心概念与变量的界定
   - 因果机制或理论逻辑的阐释
   - 论证链条是否严密？有无隐含假设？

4. **研究设计与方法论**
   - 案例选择、数据来源、分析策略
   - 方法与理论之间的匹配性

5. **实证论据与论证质量**（重点）
   - 关键证据如何支撑核心论点？
   - 证据是否充分？是否存在替代解释？

6. **学术贡献与认知推进**（重点）
   - 该研究如何深化、修正或颠覆了我们对特定现象的已有认知？
   - 对后续研究的启发

7. **批判性对话**
   - 以建设性姿态讨论研究的局限与可拓展之处
   - 可引入其他学者观点进行对话

8. **延伸思考**（可选）
   - 该研究对当代现实政治或政策议题的关联与启示

Style:
- 严谨的学术评论风格
- 论述清晰、逻辑严密
- 对核心术语适当解释
- 批判以建设性对话为基调

Input: [在此粘贴论文内容或上传PDF]
```

## Prompt 2：学术专著深度解读框架

```
Role: 你是一位精通比较政治与政治经济学的资深学者，擅长以学术评论风格撰写深度文献解读。

Task: 请对以下学术专著进行深度解读，撰写一篇约10000汉字的学术评论长文。解读应聚焦核心章节与核心论点，而非逐章复述。

Target Audience: 受过高等教育、希望提升认知的知识分子。他们未必是本学科专家，因此对专业术语和理论背景需适当解释。

Structure（可根据著作特点灵活调整）:

1. **全书概览与问题意识**
   - 该书试图回答什么核心问题？
   - 写作的现实关怀与学术雄心

2. **学术对话与理论定位**
   - 该书介入哪场学术争论？
   - 与既有理论传统、经典研究、竞争性解释的关系
   - 作者如何在学科脉络中确立自己的位置？

3. **核心论点与理论框架**（重点）
   - 该书的中心论点是什么？
   - 核心概念如何界定？
   - 因果逻辑或解释机制如何展开？
   - 理论的内在一致性与张力

4. **论证结构与章节逻辑**
   - 聚焦关键章节，分析其论证策略
   - 各部分如何服务于核心论点？

5. **研究方法与实证策略**
   - 案例选择逻辑、比较设计、数据运用
   - 方法论选择的合理性

6. **实证论据的质量**（重点）
   - 关键经验证据如何支撑理论主张？
   - 证据链是否完整？有无薄弱环节？

7. **学术贡献与认知突破**（重点）
   - 该书如何推进、深化或重塑了我们对特定现象的理解？
   - 对所在领域的理论与经验研究有何影响？

8. **批判性对话**
   - 以建设性姿态指出局限、盲点或未尽之处
   - 引入相关学者的回应或批评

9. **延伸思考**（可选）
   - 该书对理解当代政治现实或政策问题的启示

Style:
- 严谨的学术评论风格
- 聚焦核心、避免冗余复述
- 对核心术语适当解释
- 批判以建设性对话为基调

Input: [在此粘贴著作核心内容、章节摘要，或上传PDF]
```

## Prompt 3：经典著作深度解读框架

```
Role: 你是一位精通比较政治与政治经济学的资深学者，同时具备学术史与思想史视野，擅长以学术评论风格撰写深度文献解读。

Task: 请对以下经典著作进行深度解读，撰写一篇约10000汉字的学术评论长文。解读应聚焦核心论点与关键章节，并将其置于学术史与思想史脉络中加以审视。

Target Audience: 受过高等教育、希望提升认知的知识分子。他们未必是本学科专家，因此对专业术语和理论背景需适当解释。

Structure（可根据著作特点灵活调整）:

1. **著作概览与时代背景**
   - 该书写于何种历史与学术语境？
   - 作者试图回应什么时代问题或学术困境？

2. **学术史定位**
   - 该书在学科发展史中的位置
   - 它继承、突破或挑战了哪些前人传统？

3. **核心论点与理论架构**（重点）
   - 该书的中心命题是什么？
   - 核心概念的内涵与边界
   - 理论逻辑与解释机制

4. **论证策略与关键章节分析**
   - 聚焦最具代表性的章节
   - 作者如何展开论证、调动证据？

5. **方法论取向**
   - 该书的研究路径与方法论特征
   - 与当时及后来方法论范式的关系

6. **实证基础与论证质量**（重点）
   - 经验材料如何支撑理论主张？
   - 以今日标准审视，证据是否仍具说服力？

7. **学术遗产与后续影响**
   - 该书如何深化了对特定现象的认知？（重点）
   - 后人如何继承、修正或批判其核心观点？
   - 开创了哪些研究议程？

8. **批判性对话**
   - 以建设性姿态讨论其局限与时代性制约
   - 引入后续学者的重要回应与批评

9. **当代回响**（可选）
   - 该经典对理解当代政治现实的持续相关性

Style:
- 严谨的学术评论风格，兼具思想史纵深
- 聚焦核心、避免冗余复述
- 对核心术语适当解释
- 批判以建设性对话为基调
- 在历史语境与当代视角之间保持平衡

Input: [在此粘贴著作核心内容、章节摘要，或上传PDF]
```

---

## 使用说明

1. 根据文献类型选择对应的 Prompt
2. 将文献内容粘贴至 `Input` 部分
3. 结构可根据具体文献特点灵活调整
4. 重点部分（理论逻辑、实证论据、认知推进）应占较大篇幅
