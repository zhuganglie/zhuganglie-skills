# Example: Semiconductor Industry

## Example User Prompt

半导体行业是怎么演化到今天这个格局的？

## Why HDA Fits

- Subject type: technology plus industry
- Core task: explain phases, turning points, actors, and structural logic
- Likely scope choice: global trajectory with emphasis on the US-East Asia production network

## Sample Answer

半导体行业的主线，不只是“芯片越来越先进”，而是从实验室中的军工电子元件，演化成一个由设计、制造、设备、材料、EDA 和地缘政治共同塑造的全球性关键基础设施体系。它的底层变化，是从垂直一体化的企业能力竞争，转向模块化分工下的系统性瓶颈竞争。

范围：以全球路径为主，重点看美国与东亚生产网络；时间窗口从 1947 年至今；主镜头是技术能力、产业组织与地缘政治。

### 一、演化阶段

- 1947-1965，实验室与军工起源期
  从分立电子器件走向可批量制造的半导体元件。核心特征是技术仍主要服务于军工、通信和早期计算，研发集中在 Bell Labs、Texas Instruments、Fairchild 等少数机构，行业逻辑是“谁能做出来”。

- 1965-1985，集成电路商业扩张期
  从专用器件走向通用计算平台。核心特征是集成电路和微处理器推动成本下降与性能提升，芯片开始进入企业计算、工业控制和消费电子，行业逻辑变成“谁能规模化迭代并建立产品路线图”。

- 1985-2010，全球分工与代工成型期
  从垂直整合走向设计、制造、封测和设备的模块化分工。核心特征是 fabless-foundry 模式崛起，台积电等代工厂成为产业枢纽，美国强化设计与 EDA 优势，东亚强化制造与供应链协同，行业逻辑转为“谁占据链条中的关键节点”。

- 2010年代以来，算力基础设施与地缘政治期
  从单纯的商业竞争走向国家能力竞争。核心特征是先进制程、GPU、AI 加速器和高带宽封装成为战略资源，设备、软件和制造能力出现明显瓶颈，行业逻辑变成“谁控制不可替代的 chokepoints”。

### 二、关键转折点

- 1947，晶体管问世：电子系统第一次摆脱真空管的体积、功耗和可靠性约束，半导体从科学问题变成工程路径。
- 1958-1959，集成电路出现：把多个元件集成到单一芯片上后，性能提升不再依赖手工组装，行业进入真正可扩展的制造逻辑。
- 1971，商用微处理器出现：计算能力被标准化为可出售的通用部件，芯片从“设备内零件”变成“平台核心”。
- 1987，台积电成立：晶圆代工模式制度化，设计与制造分离成为可能，整个产业的组织形式被重写。
- 2010年代后期，AI 与先进制程瓶颈叠加：最先进芯片不再只是消费电子升级件，而是云计算、军事能力和国家产业政策的核心争夺对象。

### 三、塑造者与关键人物

- Bell Labs + 研究机构 + 把晶体管从理论和材料问题推进为可用器件
- Robert Noyce + 企业家兼技术推动者 + 推动集成电路商业化，并奠定硅谷式半导体创业路径
- Gordon Moore + 行业路线图制定者 + 通过摩尔定律把性能迭代预期变成整个行业的协调机制
- Morris Chang + 组织架构设计者 + 用纯代工模式重塑全球芯片产业分工
- ASML + 关键设备企业 + 把先进光刻能力变成少数企业才能掌握的产业瓶颈

### 四、底层逻辑总结

半导体行业的历史，本质上是“从器件发明竞争，走向制造规模竞争，再走向系统性瓶颈控制竞争”的历史；越往后，真正稀缺的越不是芯片本身，而是那些让最先进芯片能够被持续设计、制造并量产出来的组织能力、设备能力与供应链位置。
