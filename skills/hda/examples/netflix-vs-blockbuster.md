# Example: Netflix vs Blockbuster

## Example User Prompt

为什么 Netflix 能打败 Blockbuster？

## Why HDA Only Partly Fits

- Subject type: firm-level outcome inside a larger distribution shift
- Immediate task: explain a specific competitive result
- Better move: answer the direct causal question first, then add only the historical background that materially sharpens it

## Sample Answer

Netflix 能打败 Blockbuster，首先不是因为它“更互联网”，而是因为两家公司押注的是两套不同的分发逻辑。Blockbuster 的利润结构长期依赖门店网络、库存周转和滞纳金；Netflix 先用邮寄订阅绕开门店成本，后来又顺着宽带和终端普及切到流媒体，等于连续踩中了两次分发结构变化。

### 先直接回答原因

- 成本结构不同：Netflix 没有 Blockbuster 那样沉重的线下门店负担，更容易用订阅制换取用户留存
- 激励结构不同：Blockbuster 既得利益深绑在线下租赁和滞纳金收入上，转型会先伤到自己最稳定的现金流
- 能力路径不同：Netflix 把自己逐步建成数据驱动的平台型公司，Blockbuster 更像传统零售网络，很难在组织上快速切换
- 时点判断不同：当宽带、压缩技术和联网终端开始成熟时，Netflix 已经在为“无实体分发”做准备，Blockbuster 仍主要在旧体系里修补

### 如果补一小段历史背景，真正重要的是这三步

- 1980-1990 年代，家庭视频租赁的门店时代：核心优势是谁占据更好的零售位置、库存和渠道覆盖
- 1997-2006，邮寄订阅过渡期：Netflix 用邮寄和订阅制先拆掉门店租赁的成本与惩罚逻辑
- 2007 年以来，流媒体平台期：竞争焦点从门店网络转向内容库、推荐系统、终端入口和长期订阅关系

一句话说，Netflix 赢，不只是因为一家新公司比旧公司更灵活，而是因为视频分发从“门店控制”转向“订阅与平台控制”时，它的商业模式和组织能力更贴合新规则。
