# Example: Streaming Video

## Example User Prompt

流媒体视频平台是怎么从电视时代演化出来的？

## Why HDA Fits

- Subject type: platform plus media business model
- Core task: explain the transition from broadcast scarcity to on-demand platforms
- Likely scope choice: US-centered trajectory with global spillover

## Sample Answer

流媒体视频平台的历史，不只是“看视频更方便了”，而是内容分发权从线性排播体系转移到平台和算法体系的过程。它把影视行业从稀缺频道竞争，推向以订阅、数据、内容库和终端入口为核心的新型平台竞争。

### 一、演化阶段

- 1950年代-1990年代，广播电视与有线捆绑期
  从有限频道的线性排播体系发展出稳定的大众媒体基础设施。核心特征是入口稀缺、发行权集中、广告和有线套餐是主要收入来源，用户只能在既定时刻观看既定内容。

- 1990年代末-2000年代中后期，数字过渡与盗版冲击期
  从稳定的电视分发走向文件化、网络化传播。核心特征是宽带普及、P2P 下载和数字压缩技术削弱了传统发行时序，用户开始习惯“内容应该随时可得”，旧行业的窗口期规则被动摇。

- 2007-2018，订阅流媒体扩张期
  从下载和邮寄租赁走向随点随看的平台订阅模式。核心特征是 Netflix 等平台建立内容库加月费订阅的常态，跨终端观看成为默认体验，平台第一次直接拥有用户关系和消费数据。

- 2019年以来，平台战争与再捆绑期
  从单一订阅神话走向多平台竞争。核心特征是 Disney、Warner、Amazon、Apple 等大公司全面入场，独家内容和版权回收加剧平台分裂，行业重新出现“用新方式再捆绑”的趋势，包括广告版、体育直播、硬件入口和聚合套餐。

### 二、关键转折点

- 1999，Napster 引爆数字分发想象：虽然它主要改变的是音乐行业，但它证明了内容一旦变成文件，旧的排播和渠道控制就会被持续削弱。
- 2005，YouTube 上线：视频分发从专业机构垄断走向平台化上传与算法化分发，用户开始接受在线视频作为日常媒介形态。
- 2007，Netflix 推出流媒体服务：视频消费从“租一部片”变成“持续访问一个内容库”，订阅流媒体的商业逻辑正式成立。
- 2013，平台自制内容成熟化：当《House of Cards》这类原创内容成功后，平台不再只是渠道，而开始直接改写内容生产权力结构。
- 2019，Disney+ 上线：传统内容巨头大规模收回授权，流媒体从分发创新阶段进入版权军备竞赛阶段。

### 三、塑造者与关键人物

- Reed Hastings + 平台经营者 + 把视频消费从交易式租赁改造成订阅式持续关系
- YouTube + 平台基础设施 + 让在线视频成为低门槛、高频率的日常媒介行为
- Steve Jobs + 终端与生态协调者 + 通过 iPhone、iPad 和应用生态把“随时随地看视频”变成现实习惯
- Disney + 内容权利拥有者 + 用版权回收和独立平台战略重写流媒体竞争格局
- Roku + 终端入口构建者 + 证明客厅入口和操作系统层面的控制权同样决定平台格局

### 四、底层逻辑总结

流媒体视频平台的发展，本质上是内容分发从“频道和时间表控制”转向“平台、数据和终端控制”的过程；旧媒体时代最稀缺的是播出时段，而平台时代最稀缺的是用户注意力、版权库以及稳定的入口控制权。
