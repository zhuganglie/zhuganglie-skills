# 模块一：核心论点与理论框架

## 分析任务

### 1. 识别中心研究问题与核心论点
- 作者试图回答什么问题？
- 核心论点（thesis）是什么？用一句话概括
- **问题类型定位**：
  - **实证性（Empirical）**：解释"是什么/为什么"（比较政治/IR/行为主义）
  - **规范性（Normative）**：探讨"应当如何"（政治理论/伦理）
  - **形式化（Formal）**：基于公理的逻辑推演（博弈论/公共选择）

### 2. 提取理论框架
- **实证研究**：作者借用了哪些中层理论？提出了什么新机制？
- **规范研究**：作者基于哪些元理论（功利主义/道义论/社群主义）？核心概念如何定义？
- **形式模型**：基于哪些核心假设（理性人/信息结构）？

### 3. 理论模型处理（分类处理）

**A. 因果链条（针对实证研究）**：
```
[自变量] → [机制1] → [中介变量] → [机制2] → [因变量]
     ↑                                    
  [调节变量/边界条件]                      
```

**B. 逻辑重构（针对规范/理论研究）**：
```
[前提假设 1, 2...] → [推理步骤] → [规范性结论]
```

**C. 博弈结构（针对形式模型）**：
- **参与人**：谁在博弈？偏好序是什么？
- **策略空间**：有哪些行动选择？
- **均衡结果**：纳什均衡/完美贝叶斯均衡是什么？

**必须阐明**：
- 实证：因果机制的微观基础
- 规范：核心概念的内涵与外延
- 形式：假设的现实相关性

### 4. 理论评价维度

| 维度 | 评估问题 |
|------|---------|
| **创新性** | 相比既有理论，新在哪里？是渐进式还是范式转换？ |
| **逻辑一致性** | 假设间是否兼容？推理是否严密？ |
| **简约性** | 是否用最少假设解释最多现象？ |
| **可证伪性** | (实证/形式) 什么证据可以推翻该理论？ |
| **融贯性** | (规范) 是否符合广泛接受的道德直觉？ |

---

## 适应规则

- **若著作以理论建构为主** → 本模块权重加倍，强化逻辑一致性评估
- **若著作以实证为主** → 本模块聚焦于提取隐含理论假设
