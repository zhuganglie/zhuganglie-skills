# 模块二：研究设计与实证评估

## 方法论分析（按研究类型侧重）

### 1. 定性/定量实证研究 (Empirical)

| 类型 | 重点评估 |
|------|---------|
| **小N案例** | 案例选择逻辑（MSSD/MDSD）、跨案例比较、过程追踪证据强度 |
| **大N统计** | 变量操作化效度、数据质量、模型设定、内生性处理 |
| **混合方法** | 定性与定量的整合程度、功能分工（验证机制 vs 泛化） |

### 2. 规范性理论分析 (Normative)

| 维度 | 关键问题 |
|------|---------|
| **概念分析** | 核心概念（如"正义"、"自由"）定义是否清晰、一致？ |
| **论证逻辑** | 前提到结论的推导是否严密？是否存在滑坡谬误？ |
| **思想实验** | 假设情境（如"无知之幕"）是否合理？能否支撑结论？ |
| **竞争理论** | 是否有效回应了主要竞争对手（如功利主义 vs 罗尔斯）？ |

### 3. 形式模型/博弈论 (Formal)

| 维度 | 关键问题 |
|------|---------|
| **假设合理性** | 理性假设、信息结构假设是否过度简化现实？ |
| **解的稳健性** | 结果是否依赖于极端的参数设定？ |
| **实证检验** | 模型的预测是否生成了可检验的假设（EITM）？ |

---

## 通用评估维度（适用于实证研究）

### 因果推断策略

| 策略类型 | 评估要点 |
|---------|---------|
| 实验/准实验 | 随机化是否可信？控制组可比性？ |
| 工具变量 | 工具的外生性与相关性？排他性约束？ |
| 断点回归 | 断点处连续性？带宽选择敏感性？ |
| 双重差分 | 平行趋势假设是否成立？ |
| 匹配方法 | 匹配变量选择？隐藏偏差？ |
| 过程追踪 | 证据类型与推断强度？ |

### 效度评估

- **内部效度**：因果关系是否成立？排除了哪些替代解释？
- **外部效度**：结论可推广到哪些情境？边界条件是什么？
- **构念效度**：测量是否捕捉了理论概念？
- **统计结论效度**：统计推断是否可靠？

---

## 证据评估

- 实证证据的**充分性**：证据量是否足够支撑结论？
- 证据与论点的**匹配度**：证据是否直接检验核心假设？
- **过度推断**：结论是否超出证据支持范围？
