# 模块三：文献脉络与学术对话

## 核心任务

**不仅复述作者引用，更要主动构建学术对话网络。**

---

## 分析步骤

### 1. 被动部分：梳理作者的文献基础

- 作者显性引用了哪些核心文献？
- 作者将自己定位在哪个学术传统中？
- 作者承认受谁影响？

### 2. 主动部分：构建学术对话网络（核心）

#### 检索任务

使用 web 搜索与页面打开能力，优先检索：
- 出版社页面、作者页面、目录、导论摘要
- 学术期刊书评、review essay、symposium
- 相关领域的综述文章与 handbook 章节
- 近期引用该书或与其争论的论文

可直接使用如下检索式：
- `"[书名]" review`
- `"[作者名]" "[书名关键词]"`
- `"[核心概念]" debate political science`
- `"[子领域]" review essay handbook`

**来源优先级**：
1. 书籍与作者元数据
2. 学术书评与评论专栏
3. 综述与 handbook
4. 后续引用与修正论文
5. 一般网页背景材料（仅补充，不作主要依据）

#### 定位任务

回答以下问题：

| 对话类型 | 问题 |
|---------|------|
| **挑战** | 著作挑战了谁的理论？（实证质疑 / 规范挑战 / 模型修正） |
| **支持** | 著作支持了哪个学派？（如现实主义 vs 自由主义，罗尔斯主义 vs 社群主义） |
| **补充** | 著作填补了什么研究空白？ |
| **隐含对话** | 作者没有明确引用但实际在回应谁？ |

对“隐含对话”的判断必须谨慎：

- 若根据书中术语、争论结构、反驳对象可稳定推断，可写为“可视为在回应……”
- 若证据不足，不要把推断写成既成事实

#### 关键学者识别

识别该子领域的：
- **奠基者**：谁建立了这个研究议程？
- **主流观点**：当前学界共识是什么？
- **挑战者**：谁在挑战主流？
- **本书定位**：作者站在哪一边？

---

## 典型学术辩论模式

1. **理论竞争**：A理论 vs B理论，本书支持哪个？
2. **范围条件**：主流理论在X条件下失效，本书提供证据
3. **机制补充**：已知X→Y，但机制不明，本书揭示机制
4. **新现象发现**：发现既有理论无法解释的异常案例

---

## 输出要求

- 优先给出 2 到 4 条最重要的学术对话线，而不是罗列一长串名字
- 对每条对话线说明：对话对象是谁、争点是什么、本书站在哪里、证据基础是什么
- 区分“作者自我定位”和“你基于检索重建出的学界定位”
- 若书籍较新、书评有限，可用“相关辩论 + 初步引用链”替代完整争鸣图谱

---

## 输出格式

```markdown
## 三、文献脉络与学术对话

### 3.1 作者的文献基础
[显性引用的核心文献]
[自我定位的学术传统]

### 3.2 在核心辩论中的定位
[子领域的主要辩论]
[本书的立场]

### 3.3 与关键学者的对话
- **挑战**：挑战了 [学者X] 的 [理论Y]，因为...
- **支持**：为 [学者A] 的 [观点B] 提供了新证据...
- **补充**：填补了 [研究空白]...

### 3.4 填补的研究空白
[具体描述空白及本书贡献]
```
