# 模块四：深度批判性评价

## 核心原则

**批判必须具体，禁止空泛表述。每个批判必须配套替代方案。**

---

## 贡献识别（四层面）

| 层面 | 评估问题 |
|------|---------|
| **理论贡献** | 提出了什么新概念/新机制/新框架？ |
| **实证贡献** | 提供了什么新数据/新发现/新证据？ |
| **方法论贡献** | 引入了什么新方法/新设计/新测量？ |
| **政策/实践贡献** | 对政策制定者/实践者有什么启示？ |

---

## 局限性批判（必须具体）

### 理论层面（实证/规范/形式）

| 问题类型 | 批判要点 |
|---------|---------|
| **假设问题** | (实证) 假设不现实；(规范) 道德直觉不可靠；(形式) 假设过度简化 |
| **概念问题** | 定义模糊、概念拉伸（conceptual stretching）、内涵不一致 |
| **逻辑问题** | (实证) 因果机制断裂；(规范) 推理跳跃；(形式) 数学推导错误 |
| **适用性** | (实证) 外部效度低；(规范) 忽视现实约束（理想理论问题） |

### 方法论层面

| 问题类型 | 批判要点 |
|---------|---------|
| 选择偏差 | 案例/样本选择是否有系统偏差？ |
| 测量误差 | 变量操作化是否有效？ |
| 遗漏变量 | 是否忽略了重要的混淆因素？ |
| 内生性 | 是否存在反向因果或共同原因？ |
| 外部效度 | 结论能否推广到其他情境？ |

### 证据层面

| 问题类型 | 批判要点 |
|---------|---------|
| 证据不足 | 哪些核心论点缺乏充分证据？ |
| 选择性呈现 | 是否只展示支持性证据？ |
| 过度推断 | 结论是否超出证据支持范围？ |

---

## 替代方案（必须提供）

### 替代性解释

- **反向因果**：Y 是否可能导致 X？
- **共同原因**：是否存在 Z 同时影响 X 和 Y？
- **遗漏机制**：是否存在未考虑的中介路径？
- **调节效应**：效果是否因情境而异？

### 理论修正方向

- 如何放松不现实的假设？
- 如何澄清模糊的概念？
- 如何扩展/限缩适用边界？

### 方法改进建议

- 更好的案例选择策略是什么？
- 更有效的测量方法是什么？
- 如何解决内生性问题？
